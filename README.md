# Features

# Installation Requirements
* Do some git stuff
* node and npm
* nodemon

```
git clone https://github.com/xMTinkerer/octoapp.git
cd octoapp
npm install
```

# Run
Per the `package.json`, just fire up:
```
npm start
```

Then navigate your browser to `http://localhost:9988` and pick a button.
